# Timer

A simple timer web application built using *HTML, **Tailwind CSS, and **JavaScript*.
It helps track time easily with start, stop, and reset functions.

## Features

* Start the timer
* Stop / Pause the timer
* Reset the timer
* Simple and clean user interface

## Technologies Used

* HTML
* Tailwind CSS
* JavaScript

## How to Use

1. Open the project in your browser.
2. Click *Start* to begin the timer.
3. Click *Stop* to pause the timer.
4. Click *Reset* to set the timer back to zero.

## Purpose

This project was created to practice basic web development using HTML, Tailwind CSS, and JavaScript.
